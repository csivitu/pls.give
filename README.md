# pls.give

## Usage
Please refer to the ``sample.env`` file and fill in the values accordingly. 

## Features
1. GG URL shortener
``pls.give link http://some-big-link.com``
<br />

2. Whiteboard 
``pls.give board``
<br />

3. Pastebin 
```pls.give paste long text that can be separated by spaces```
<br />
